# @holocronlab/botruntime-zui

A fork of Zod with additional features (JSON Schema import/export, UI hints,
extended validators). This package is a direct fork of
[`@bpinternal/zui`](https://github.com/botpress/botpress) (MIT licensed),
rebranded under the `@holocronlab` scope for use by `botruntime` packages
(notably `@holocronlab/botruntime-sdk`). Its public API and exports are kept
identical to upstream `zui` so it is a drop-in replacement.

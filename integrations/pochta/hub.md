# Почта России

Глобальная интеграция для чтения истории операций регистрируемого почтового
отправления через официальный SOAP API Сервиса отслеживания Почты России.

## Настройка

- `login` — логин из вкладки «API Трекинга»;
- `password` — пароль API, хранится как secret конфигурации.

На странице настроек Почты России есть две разные вкладки. Эта интеграция
использует SOAP API из вкладки «API Трекинга». Не вводите `AccessToken`
из вкладки «API Почта Бизнес»: он предназначен для другого REST API и не
подходит для регистрации интеграции.

Интеграция работает с внутрироссийскими 14-значными идентификаторами и
международными 13-символьными идентификаторами S10. Использовать API можно только
для отправлений, которые пользователь вправе отслеживать по условиям сервиса.
Ограниченный доступ допускает до 100 единичных запросов в сутки — частоту polling
должен задавать вызывающий бот.

## Action

- `trackShipment` — возвращает нормализованную историю операций и состояние
  `not_found`, `in_transit`, `delivered` или `returned`. Вручение определяется по
  `OperType=2`, возврат — по `OperType=3`; исходные коды, названия и время каждой
  операции сохраняются в ответе для аудита.
